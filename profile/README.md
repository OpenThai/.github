# OpenThai

เราให้ความสนใจกับการพัฒนา **Personal AI**, **AI Agent**, **AI Coding Agent**, และโครงสร้างพื้นฐานสำหรับระบบอัจฉริยะที่สามารถเข้าใจบริบท ใช้เครื่องมือ ทำงานร่วมกับมนุษย์ และช่วยให้การทำงานมีประสิทธิภาพมากขึ้น

---

## สิ่งที่เราสนใจ

OpenThai มุ่งศึกษาและพัฒนาในด้านต่าง ๆ เช่น:

- **Personal AI Assistant**  
  ผู้ช่วย AI ส่วนตัวที่เข้าใจบริบทของผู้ใช้ จดจำข้อมูลสำคัญ จัดการความรู้ งาน เป้าหมาย และเวิร์กโฟลว์ระยะยาวได้

- **AI Agent**  
  ระบบ AI ที่สามารถวางแผน ใช้เครื่องมือ ค้นคว้า วิเคราะห์ ตัดสินใจ และทำงานเป็นขั้นตอนได้อย่างปลอดภัย

- **AI Coding Agent**  
  ผู้ช่วยเขียนโค้ดที่สามารถอ่านโปรเจกต์ เข้าใจโครงสร้างโค้ด แก้ไขไฟล์ รันคำสั่ง ตรวจสอบผลลัพธ์ และอธิบายการเปลี่ยนแปลงได้

- **Agentic Software Development**  
  การพัฒนาซอฟต์แวร์ด้วย AI Agent ที่ทำงานร่วมกับนักพัฒนา ตั้งแต่การวางแผน เขียนโค้ด ทดสอบ ตรวจสอบ ไปจนถึงสรุปผล

- **Memory & Context System**  
  ระบบความจำและบริบทสำหรับ AI เช่น project memory, knowledgebase, retrieval system และ context engineering

- **AI Infrastructure**  
  โครงสร้างพื้นฐานสำหรับรัน AI Agent เช่น runtime, tool system, provider routing, plugin, verification และ multi-agent workflow

---

## เป้าหมายของเรา

OpenThai ต้องการสร้างเทคโนโลยี AI ที่:

- รองรับภาษาไทยได้ดีขึ้น
- เข้าใจบริบทของผู้ใช้และโปรเจกต์
- ทำงานร่วมกับมนุษย์อย่างโปร่งใส
- ช่วยลดงานซ้ำซ้อน
- เพิ่มความสามารถในการคิด วิเคราะห์ และสร้างสรรค์
- ใช้งานได้จริงในงานส่วนตัว งานธุรกิจ และงานพัฒนาซอฟต์แวร์

เราเชื่อว่า AI ไม่ควรเป็นเพียงแค่แชตบอตตอบคำถาม แต่ควรเป็นระบบที่สามารถช่วยคิด ช่วยทำงาน ช่วยตรวจสอบ และช่วยพัฒนางานร่วมกับมนุษย์ได้อย่างเป็นระบบ

---

## แนวทางการพัฒนา

OpenThai ให้ความสำคัญกับหลักการต่อไปนี้:

- **Human-in-the-loop**  
  AI ควรช่วยมนุษย์ตัดสินใจ ไม่ใช่ทำงานแทนโดยไม่มีการตรวจสอบ

- **Local-first when possible**  
  ข้อมูลสำคัญควรอยู่ใกล้ผู้ใช้และโปรเจกต์มากที่สุดเท่าที่เป็นไปได้

- **Transparent Workflow**  
  แผนงาน การกระทำ การใช้เครื่องมือ และผลลัพธ์ของ AI ควรตรวจสอบย้อนหลังได้

- **Verification Matters**  
  งานที่ AI สร้างควรถูกทดสอบ ตรวจสอบ และยืนยันผลลัพธ์ก่อนใช้งานจริง

- **Composable System**  
  Agent, tool, memory, model และ plugin ควรถูกออกแบบให้ประกอบและขยายต่อได้ง่าย

## สถานะ

OpenThai อยู่ในช่วงการวิจัย ทดลอง และพัฒนา

เรากำลังศึกษาแนวทางใหม่ ๆ ในการสร้าง AI ที่เข้าใจบริบท ทำงานร่วมกับมนุษย์ ใช้เครื่องมือได้ดี และสามารถช่วยพัฒนาซอฟต์แวร์หรือจัดการงานซับซ้อนได้อย่างปลอดภัย

## วิสัยทัศน์

**OpenThai ต้องการเป็นส่วนหนึ่งของการสร้าง AI ที่เข้าใจภาษาไทย เข้าใจผู้ใช้ และช่วยให้มนุษย์ทำงานได้ดีขึ้น**

เราเชื่อว่าอนาคตของ AI จะไม่ใช่แค่ระบบถามตอบ แต่คือระบบผู้ช่วยอัจฉริยะที่สามารถเรียนรู้บริบท ใช้เครื่องมือ วางแผน ทำงาน และเติบโตไปพร้อมกับผู้ใช้
